## 🚀 About Me

<img align="right" alt="Coding" width="320" src="https://raw.githubusercontent.com/abhisheknaiidu/abhisheknaiidu/master/code.gif"/>

I'm **Swayam Gunjal** — a developer who believes the strongest apps are built on the strongest fundamentals. Instead of chasing frameworks first, I've been deliberately grinding through **C, C++, Java, Python, and JavaScript**, treating every repo as a rep at the gym for my brain.

🔭 Currently sharpening core problem-solving skills across multiple languages
🌱 Learning how the *same* problem looks different in *different* paradigms
🎯 Goal: turn strong fundamentals into real, shippable projects
⚡ Fun fact: I enjoy restarting in a new language just to see how it "thinks"
